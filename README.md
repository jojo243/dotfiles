# dotfiles

My dotfiles repository, managed by yadm ([yadm.io](https://yadm.io/)).

## How to

Install yadm:

```bash
sudo apt-get install yadm
```

Bootstrap:

```bash
yadm clone https://github.com/jojo243/dotfiles
yadm status
```
