# dotfiles

My dotfiles repository, managed by yadm ([yadm.io](https://yadm.io/)).

## How to

Install yadm:

```bash
sudo apt-get install yadm
```

Bootstrap:

[yadm bootstrap](https://yadm.io/docs/bootstrap)
